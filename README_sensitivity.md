# DDM Sensitivity Analysis – Fixed vs. Dynamic Required Return (r)

Supplementary analysis for the bachelor's thesis:  
**Timing of Stock Purchases on the Prague Stock Exchange**  
Nick Trefil, 2026

---

## What this script does

The main backtest uses a **fixed required rate of return r = 9 %** in the Gordon
Growth Model across all 21 years (2004–2024). This analysis tests how sensitive
the DDM undervaluation verdict is to using a **dynamic r** derived from actual
10-year Czech government bond yields plus a 5 % equity risk premium.

All three files (`sensitivity_analysis.py`, `pomocná_data.xlsx`, `dluhopisy.xlsx`)
must be in the same folder.

## Output

The script generates `sensitivity_output.xlsx` with three sheets:

- **Detail** – full row-level results for every ticker and year
- **Souhrn** – year-by-year summary (suitable for the thesis table)
- **Změny** – only rows where the DDM verdict differs between fixed and dynamic r

---

## Key findings

| | |
|---|---|
| Total observations | 123 |
| Verdict changes (fixed → dynamic r) | 27 (22.0 %) |
| Dynamic r range | 5.5 % – 9.8 % |
| Dynamic r average | 7.8 % |

**All 27 changes go in one direction: NE → ANO.**

The fixed r = 9 % is systematically more conservative than the market-implied r
in low-interest-rate years (approx. 2013–2021), meaning it *disadvantages* the
strategy rather than inflating its results. In years where bond yields were close
to their long-term average (2004–2012, 2023–2024), the two approaches produce
identical verdicts.
